# REPO ModPack

## Download

Download the zipped file from this repository.

## Installation

1. Open Steam and go to your **REPO** game.
2. Right-click → **Properties** → **Browse...** to open the game folder.
3. Unzip the downloaded file into the game folder. Example path:  
<Steam Location>\Steam\steamapps\common\REPO


## Requirements

For the modpack to work correctly, you need:

- **BepInEx**
- **winhttp.dll**
- **doorstop_config**

Make sure these are installed in the game folder before running.
